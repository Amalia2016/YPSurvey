Missing Values

    View details »

  • In case of categorical features with missing values, observations were dropped.

  • In case of ordinal features with missing values, value was imputed using the mean.

Outliers

    View details »

  • Only interval features such as age, height and weight, presented outliers; very few, not extreme nor significant for the purpose of this analysis.

Transformations

    View details »

  • Categorical features were encoded using pandas get_dummies.

  • Behavioural features of interest (Finances/Savers, Smoking and Alcohol) were transformed to binary for modeling (binary class prediction).

Correlations

  • Correlations within and among categories were explored.

  • At least for savers, personality traits seemed to have greater influence than other categories of information.

Feature: Finances/Savers

    View details »

  • Correlations provided some idea of the features that might influence more a saver's behaviour.

  • Those features would be used to model and predict who is a saver.
